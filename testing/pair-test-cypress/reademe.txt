Для запуска тестов необходимо: 
Открыть проект, через терминал установить все пакеты и зависимости командой npm i 
Для теста указан адрес который открывает serve cy.visit('http://localhost:3000') 
Заустить сервер или при необходимости изменить адрес входа в cypress, e2e, pair-test.cy.js
Запустить тесты командой npm run test 
В открывшемся окне нажать Configured
Выбрать браузер и нажать start e2e test
В specs выбрать pair app
