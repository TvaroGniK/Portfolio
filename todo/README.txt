Приложение имеет два варианта работы, с серверным хранилищем и с локальным(LocalStorage)

Для запуска приложения:
- откройте в редакторе todo-server и в корне проекта в терминале запустите сервер командой node index
- откройте в редакторе todoApp изапустите liveserver  в корне проекта.

Вы можете довалять дела, удалять и переводить в состояние готовности. После перехагрузки страницы все дела и их состояния сохраняются 

В проетке реализована модульная структура, для упрощения разработки.
