Для запуска проекта необходимо:
1 Иметь установленную версию node.js 
2 Открыть содержимое "crm-backend" в редакторе кода и через терминал запустить сервер командой node index
3 Открыть содержимое "clients" в редакторе кода и через live server ( или serve) запустить проект 
в проекте выполнениы все основные и дополнительные пункты техническогозадания
